# Glossaries

All glossaries use the same JSON format: `{ "source_term": "target_term" }`.

| File | Direction | Method | Terms | Used for |
|------|-----------|--------|-------|----------|
| `eu_parliament_es.json` | ES keys only | Manual | ~27 | Whisper `initial_prompt` (ASR domain adaptation) |
| `eu_parliament_es_en.json` | ES → EN | Manual | ~25 | Qwen system prompt (translation domain adaptation) |
| `eu_parliament_es_en_tfidf.json` | ES → EN | TF-IDF auto | ~200 | Exp 4 comparison (generated by `scripts/extract_glossary_tfidf.py`) |
| `eu_parliament_es_en_llm.json` | ES → EN | LLM-generated | ~100 | Exp 4 comparison (generated by `scripts/extract_glossary_llm.py`) |

The TF-IDF and LLM glossary files are not committed — generate them with the scripts above after preparing the EPIC corpus.
